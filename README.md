# UselessDiscordBot

## Install 
`npm install node.js`

`npm install googleapis`

`npm install dotenv`


## Run

`node index.js`

Bot will show as 'Online' on Discord and run until Ctrl+C

## Reload command
`!reload <command_name>`

If editing index.js, or any file outside the 'commands' folder, then Ctrl+C and run bot again
